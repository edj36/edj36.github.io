OCaml # OCaml Client/Server Recently in a project for school, my team and I decided to build a server to complement the game we built. The [course][10x] is about functional programming and is taught in OCaml, thus we needed to use Ocaml to implement the client and server we wanted for our game. I certainly struggled at first finding useful examples or documentation for my needs, so here I am going to try and make my own. ## Setup * * * First, install Ocaml, [here][install] is how I did it. Now that that is up and running, create a new project and create a Makefile as follows: ``` client: ocamlbuild -pkg cohttp.lwt client_example.native && ./client_example.native server: ocamlbuild -pkg cohttp.lwt server_example.native && ./server_example.native clean: ocamlbuild -clean ``` But wait! Whats `cohttp` and `lwt`? Those are the libaries we are going to install for networking. Get them from `opam` like this: ``` $ opam install cohttp lwt ``` Specifically, `cohttp` is a popular OCaml HTTP library, and `lwt` (Light Weight Threading) is exactly that, a light-weight thread library for Ocaml. ## Client * * * More to come ## Server * * * More to come [install]: http://www.cs.cornell.edu/courses/cs3110/2016fa/install.html [10x]: http://www.cs.cornell.edu/courses/cs3110/2016fa/index.php
